## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm, yarn, pnpm, or bun
- Git

### Development Setup

1. **Clone the repository**:

   ```bash
   git clone https://github.com/fusion-calling/my-app.git
   cd my-app
   ```

2. **Install dependencies**:

   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Run the development server**:

   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. **Open your browser**:
   ```
   http://localhost:3000
   ```

The app will auto-reload as you make changes.

### Build for Production

```bash
npm run build
npm start
```
